# Financial Performance & Market Share Dashboard – Power BI

# Project Overview
This project analyzes financial performance of Nestlé India, Dodla Dairy, and Dangee Dums using publicly available financial data (2019–2025).

The dashboard evaluates revenue growth, profitability trends, market share distribution, and financial KPIs.

# Data Source
Financial data compiled from publicly available company financial reports via Screener.in and Moneycontrol.
This project is created for educational and analytical purposes only.

# Dashboard Features
- KPI Cards (Revenue, Net Profit, Total Assets, EPS)
- Revenue Trend with Forecast
- Net Profit Trend Analysis
- Market Share Comparison by Product Category
- Product Category Revenue Contribution
- Interactive Company Filter

# Tools Used
- Power BI
- DAX
- Data Modeling
- Financial Ratio Analysis
